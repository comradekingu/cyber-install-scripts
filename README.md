# Cyber Install Scripts

This contains CyberOS-specific installation scripts used by Calamares.
Derived from [arch-install-scripts](https://git.archlinux.org/arch-install-scripts.git).

## Requirements

* GNU coreutils (>= v8.15)
* util-linux (>= 2.23)
* POSIX awk
* bash (>= 4.1)
